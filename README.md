# 🏥 Rural Telemedicine System

A telemedicine-based digital healthcare platform designed to provide timely medical services to rural populations using AI, real-time communication, and remote consultation.

---

## 🚀 Features

* 👨‍⚕️ Online Doctor Consultation
* 💬 Real-time Communication
* 🧠 AI-based Health Assistance
* 📋 Patient Health Records Management
* 📍 Nearby Hospitals Information
* 🩺 Symptom Checker

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Python (Flask / FastAPI)
* Node.js (if used)

### Other Tools

* Git & GitHub
* Arduino (for hardware integration)

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/sreehithaguduru16/rural-telemedicine-system.git
cd rural-telemedicine-system
```

---

### 2️⃣ Setup Frontend

```
cd frontend
npm install
npm start
```

---

### 3️⃣ Setup Backend

```
cd backend
pip install -r requirements.txt
python app.py
```
